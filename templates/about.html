Purpose of this Site:

Thank you for checking out Player Picker Pro! This site was designed by Pete Stewart in order to practice and demonstrate a few skills using python for data management. The site was built completely with python using the flask framework. The soccer player data is contained in an SQLite database that was compiled using several publicly available CSV data files that were constructed as an exercise in the use of SQL statements with python.
